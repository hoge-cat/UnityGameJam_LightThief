# Low-Poly Western Starter Pack - Documentation

Thank you for purchasing the “Low-Poly Western Starter Pack”!
This package includes 3 high-quality low-poly characters that are ready to use in your games.

---------------------------------------------------------------------
[ 1. Contents ]
- Prefabs: 3 characters (Gun man, etc.)
- Models: FBX format, with humanoid rigs
- Textures: Includes color textures

[ 2. How to Use ]
1. Open the “Prefabs” folder in the Project window.
2. Drag and drop the character prefabs directly into your scene.
3. To apply animations to the characters, use Unity’s standard humanoid animation system (Animator Controller).

[ 3. Technical Information ]
- Render Pipeline: Built-in (Standard Shader)
- Rig: Generic / Humanoid
